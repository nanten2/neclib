neclib.parameters.obsparam¶

interval(quantity, *, unit, points_per_scan=None)[source]¶

Parse interval.

Parameters

  • quantity (Quantity) – Input value to be parsed.

  • unit (Union[str, Unit]) – Unit in which return value is given.

  • points_per_scan (Optional[int]) – Number of observation points per 1 scan line.

Return type

Tuple[float, str]

Examples

>>> interval(Quantity("5min"), unit="s")
300, "time"
>>> interval(Quantity("5scan"), unit="point", points_per_scan=5)
25, "point"
off_point_coord(*, unit, on_point=None, offset=None, coslat_applied=None, absolute=None, **kwargs)[source]¶

Parse OFF observation coordinate.

Give (absolute coordinate of OFF point, output unit) or (coordinate of on_point, offset from the ON point, if the offset value has coslat_applied, output unit). The coordinates should be given in the form: (longitude, latitude, coordinate frame)

Parameters

  • on_point (Optional[Tuple[Quantity, Quantity, Union[str, BaseCoordinateFrame]]]) – Coordinate of ON point.

  • offset (Optional[Tuple[Quantity, Quantity, Union[str, BaseCoordinateFrame]]]) – Offset from ON point to OFF point. Caution the unit, as Angle("0h0m1s") is not equivalent to Angle("1arcsec").

  • coslat_applied (Optional[bool]) – If False, longitude of offset will be divided by cos(latitude).

  • absolute (Optional[Tuple[Quantity, Quantity, Union[str, BaseCoordinateFrame]]]) – Absolute coordinate of OFF point.

  • unit (Union[str, Unit]) – Unit in which return value is given.

  • **kwargs – Keyword arguments to SkyCoord, will be passed to coordinate frame class such as AltAz; location, obstime, obswl, etc.

Return type

Tuple[float, float, str]

Examples

>>> off_point_coord(
...     absolute=(Angle("3h25m20s"), Angle("31d32m33s"), "fk5"),
...     unit="deg",
... )
(51.3333, 31.5425, "fk5")
>>> off_point_coord(
...     on_point=(Angle("5h1m1s"), Angle("30d25m20s"), "galactic"),
...     offset=(Angle("0 0 10 hours"), Angle("20arcsec"), "fk5"),
...     coslat_applied=True,
...     unit="deg",
... )
(266.6442444454571, 48.39027455272732, 'fk5')
